Bonus Flexbox Session

There are two sets of CSS properties in the Flexbox system.

  1. One set for the parent, aka the "flex-container"
  2. One set for the children, aka the "flex-items"

Flex-Container Properties

The parent tells its children how to behave.

  1. display: flex
    This property set to flex will change any element into a flex container. 
  2. justify-content
    Defines the layout of flex-items in a flex-container on the main axis (x-axis). 
  3. align-items
    Defines the layout of flex-items in a flex-container on the cross axis (y-axis). 
  4. gap
    Defines the minimum spacing between flex-items.

The moment you flex an element, every direct child of that element becomes a flex-item.

Flex-Item Properties

  1. align-self
    allows a child to misbehave. 
  2. flex
    defines how much of the parent a child is allowed to eat
